Windows setup

Download. Activate. Keep control.

ClickAssist is a native Windows desktop app. The click engine runs locally; licensing is verified by the ClickAssist server.

Install

Download the Windows installer, open it, and follow the installer prompts. Windows SmartScreen may warn until the app has code signing and reputation history.

Download installer

Activate

Open ClickAssist, press Activate, enter the purchase email and license key, then arm the console. The license token is stored locally and refreshed periodically.

Safety Controls

Use F8 to stop the engine immediately. If Background mode is enabled, closing the window hides ClickAssist instead of exiting; launching ClickAssist again brings the same running instance back.
